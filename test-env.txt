# Test Environment

## Setup

1. Create VirtualBox machine.

2. Install Debian Linux OS.
	- Create user `pi`.
	- Software:
		- `standard system utilities`

3. Install the Guest Additions
	- https://virtualboxes.org/doc/installing-guest-additions-on-debian/

4. Install software packages:

	```bash
	sudo apt-get update && sudo apt-get -y dist-upgrade
	sudo apt-get -y install \
		bash-completion \
		git \
		vim \
		python3 \
		python3-pip
	```

5. Install Python modules:

	```bash
	sudo python3 -m pip install pygame pytmx pyscroll
	```

6. Clone Git project:

	```bash
	sudo mkdir /usr/local/bin/robot
	sudo chown pi:pi /usr/local/bin/robot
	git clone https://github.com/synadrin/robot.git /usr/local/bin/robot
	```

7. Setup auto-login:

	```bash
	cat /lib/systemd/system/getty@.service \
		| sed -e 's/ExecStart=-\/sbin\/agetty --noclear %I $TERM/ExecStart=-\/sbin\/agetty --noclear -a pi %I $TERM/;' \
		> /tmp/newgetty
	sudo cp /tmp/newgetty /lib/systemd/system/getty@.service
rm /tmp/newgetty
